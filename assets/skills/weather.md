# 天气查询

## 能力
- 查询当前天气
- 获取天气预报
- 提供穿衣建议

## 使用方式

### 查询天气
当用户询问天气时：
1. 使用 `web_search` 搜索 "[城市名] 天气 今天"
2. 或使用 `web_fetch` 抓取天气网站内容
3. 整理成简洁的天气信息回复

### 回复格式
```
🌤 [城市] 天气
温度：XX°C ~ XX°C
天气：晴/多云/雨...
风力：X级
穿衣建议：...
```

## 注意事项
- 需要联网才能获取实时天气
- 如果用户没有指定城市，询问所在城市
- 可以结合用户偏好提供个性化建议
