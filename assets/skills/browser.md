# 网页搜索与抓取

## 能力
- 使用 web_search 工具搜索互联网信息
- 使用 web_fetch 工具抓取网页全文内容

## 使用方式

### 搜索信息
当用户要求搜索、查找、查询互联网信息时：
1. 使用 `web_search` 工具，传入 `query` 参数
2. 返回搜索结果列表（标题、链接、摘要）
3. 如果需要详细内容，用 `web_fetch` 抓取具体页面

### 抓取网页
当用户要求查看网页内容、获取在线资料时：
1. 使用 `web_fetch` 工具，传入 `url` 参数
2. 返回网页纯文本内容（去除HTML标签）

## 注意事项
- 搜索结果可能不准确，建议抓取原文确认
- 某些网站可能无法抓取（需要登录、有反爬机制）
- 抓取内容默认截断到5000字符，可通过 max_length 参数调整
