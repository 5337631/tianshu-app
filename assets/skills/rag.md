# RAG 搜索

## 能力
- 搜索本地记忆库
- 关键词+全文+标签混合搜索
- 基于搜索结果的 AI 回答

## 使用方式

### RAG 搜索问答
在 RAG 搜索页面输入问题：
1. HybridSearchService 自动搜索记忆库
2. 找到相关记忆内容
3. 将记忆作为上下文发给 AI
4. AI 基于记忆内容回答

### 搜索原理
```
用户问题 → 分词 → 关键词匹配 + 全文匹配 + 标签匹配 → 相关度排序 → Top 5 → AI 回答
```

## 与普通对话的区别
- 普通对话：AI 用自己的知识回答
- RAG 搜索：AI 用「你的知识」回答

## 注意事项
- 搜索结果按相关度排序
- 相关度 > 30% 的结果才会被使用
- 记忆库需要有足够的内容才能发挥作用
