# 通知栏摘要

## 能力
- 读取通知栏消息
- 按应用分组
- AI 智能分类（待回复/待处理/仅供参考）
- 生成结构化摘要

## 使用方式

### 获取消息摘要
当用户要求查看消息、通知摘要时：
1. `NotificationSummaryService` 自动读取通知栏
2. 按应用分组（微信、邮件、短信等）
3. AI 分类为"待回复"、"待处理"、"仅供参考"
4. 生成结构化摘要

### 回复格式
```
📊 消息摘要

📩 **待回复 3 条**
  · [微信] 张三: 今晚吃饭？
  · [微信] 工作群: 明天开会

📋 **通知事项 2 条**
  · 快递已到菜鸟驿站
  · 订阅即将到期

💡 建议优先处理待回复项
```

## 注意事项
- 需要通知监听权限
- 首次使用需在设置中授权
- 部分应用可能不推送通知
